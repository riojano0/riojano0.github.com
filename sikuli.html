Sikuli

"Sikuli es el resultado de un trabajo de varios investigadores del MIT, con una idea detrás relativamente sencilla pero bastante espectacular.
Este IDE nos permite crear programas propios que interactúen con nuestro ordenador como una persona normal, de tal manera que puede hacer virtualmente cualquier cosa"

Este IDE nos permite por ello realizar varias automatizaciones para testear. Algunas automatizaciones que hice fueron para

Word 2013, Winzip, Winrar, Chrome
Firefox, Internet Explorer, Keepass 1.29 y 2.3
PaintNet, Process Monitor,Process Explorer, Pycharm 3.5
